# Tron Labs Website

Official website for [Tron Labs](https://mytronlabs.com) — building the data backbone for Physical AI.

---

## Tech Stack

- **Next.js 15** — framework
- **TypeScript** — language
- **Tailwind CSS** — styling
- **Vercel** — hosting

---

## Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/MaitriSavalia/tronlabs-website.git
cd tronlabs-website
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

---

## Making Changes

Edit any file inside `src/`. The browser auto-refreshes as you save.

Key files:
- `src/app/page.tsx` — Home page
- `src/app/about/page.tsx` — About page
- `src/app/solution/page.tsx` — Solution page
- `src/app/partners/page.tsx` — Partners page
- `src/app/careers/page.tsx` — Careers page
- `src/app/contact/page.tsx` — Contact page
- `src/components/Navbar.tsx` — Navigation bar
- `src/components/Footer.tsx` — Footer
- `src/app/globals.css` — Global styles and fonts

---

## Deploying to Vercel

The site auto-deploys every time you push to the `main` branch on GitHub.

### Push changes

```bash
git add .
git commit -m "your message here"
git push origin main
```

Vercel picks it up automatically. Deployment takes about 1–2 minutes.

### Check deployment status

Go to [vercel.com](https://vercel.com) → your project → Deployments tab.

---

## Build for production locally

```bash
npm run build
npm run start
```
